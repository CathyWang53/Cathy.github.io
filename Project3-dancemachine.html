A dance machine that is more interesting and know you better.

1

Overall Introduction

The magic dance machine can use hand and foot gestures to score, and adjust music speed to the heart rate.

2

Interface Design

1. Voice Input Configuration

To decrease the workload of writing codes when adding a voice input, we plan to automatically generate the codes in Geno. Developers only need to offer some examples to help train the natural language understanding model and configure multimodal interactions by dragging objects.

2. Predetermined Functions

To further decrease the workload, we predtermined some frequently-used functions that developers can select to add by one click.

For example, some functions can be used in every websites, such as scroll up/down and view account. Also, different website categories have their common functions, such as shopping sites -- view cart, media players -- play and pause, and map -- direction from A to B.

3. UI Suggestions

To help some developers who may lack design experience, we predetermined some user interface design patterns. Developers can select and customize the settings.
